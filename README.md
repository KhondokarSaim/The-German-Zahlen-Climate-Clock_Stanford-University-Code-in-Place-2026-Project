🌍 German "Zahlen" Climate Clock

An interactive educational game that combines **German language learning** with **climate change awareness**.

Created as my final project for **Stanford University's Code in Place 2026**, this project demonstrates how programming can be used to connect personal interests with real-world topics in a meaningful and engaging way.

📖 About the Project

While learning Python through Stanford University's **Code in Place 2026** program, I was also exploring German language learning and researching environmental science topics.

Instead of building a generic final project, I wanted to create something that reflected my own interests. This inspired the idea of combining both passions into a single interactive experience:

💡 The Idea

What if learning German numbers could become part of a climate-awareness game?

The result was the **German "Zahlen" Climate Clock**, an interactive game where players translate German numbers to help save a research station from rising floodwaters.

The project aims to:

- Make German number learning more engaging and enjoyable
- Introduce climate change concepts through gameplay
- Demonstrate how programming can support educational experiences

🎮 Gameplay

Players are presented with German numbers and must provide the correct translations.

Each correct answer helps protect the research station, while mistakes allow floodwaters to rise. The game combines learning, problem-solving, and environmental storytelling into a single interactive experience.


🛠 Python Concepts Used

This project applies several concepts learned throughout the Code in Place course, including:

- ✅ Python Dictionaries
- ✅ String Manipulation
- ✅ Mouse-Click Tracking
- ✅ Interactive Game Logic

🚀 What I Learned

Developing this project helped me:

- Apply core Python programming concepts in a practical setting
- Design an educational game experience
- Connect language learning with environmental education
- Transform a challenging German grammar concept into an engaging interactive activity

Most importantly, it showed me that programming is not just about writing code—it's about turning ideas, interests, and real-world challenges into meaningful solutions.
